# JumpHost

**PHP域名跳转管理系统** - 专业的域名重定向和管理平台

## 项目概述

JumpHost是一个基于PHP的域名跳转管理系统，提供灵活的域名重定向、访问统计和模板定制功能。系统无需框架，采用纯PHP开发，具有轻量级、高性能的特点。

## 核心功能

### 🌐 域名管理
- 支持多域名绑定和管理
- HTTP/HTTPS协议支持
- 批量域名跳转配置
- 域名状态控制（活跃/暂停）

### 📊 数据统计
- 实时访问统计和趋势分析
- IP地理位置识别
- 设备类型分布（手机/平板/桌面）
- 访问来源分析

### 🎨 模板系统
- 多种预设跳转模板
- 自定义模板支持
- 模板字段配置
- 响应式设计

### 👥 用户管理
- 三级权限系统（超级管理员/代理/个人用户）
- 域名所有权隔离
- 操作日志记录
- CSRF安全保护

### 🔧 系统特性
- 无需框架，纯PHP开发
- PDO数据库操作
- 完善的错误处理
- IP地理位置查询
- 验证码支持

## 安装指南

1. **环境要求**
   - PHP ≥ 7.4
   - MySQL/MariaDB
   - PDO扩展
   - mbstring扩展

2. **安装步骤**
   - 上传文件到服务器
   - 访问 `/install` 目录
   - 按照向导完成安装
   - 安装后删除 `install/` 目录

3. **默认账号**
   - 用户名: `admin`
   - 密码: `password`
   - **安装后请立即修改密码！**

## 使用说明

### 域名配置
1. 登录后台（`/admin`）
2. 添加域名规则
3. 配置目标URL和模板
4. 设置跳转参数

### 模板定制
- 模板文件位于 `templates/` 目录
- 支持 `@label` 和 `@fields` 注解
- 可自定义HTML/CSS/JavaScript

### 安全建议
- 定期更新管理员密码
- 删除安装目录
- 配置合适的文件权限
- 启用HTTPS访问

## 技术架构

- **语言**: PHP
- **数据库**: MySQL/MariaDB
- **前端**: 原生HTML/CSS/JavaScript
- **模板**: 原生PHP模板引擎

## 文件结构

```
/
├── index.php          # 主入口
├── admin/             # 后台管理
├── config/            # 配置文件
├── templates/         # 跳转模板
├── install/           # 安装程序
└── .htaccess          # URL重写规则
```
